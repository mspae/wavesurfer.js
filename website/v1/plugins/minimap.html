Version 1: Minimap plugin

Adds a minimap to the main waveform.

Quick start

See the Pen WaveSurfer.js - waveform minimap by Enton Biba (@entonbiba) on CodePen.

  1. Add the Minimap plugin script tag
  2. Create a WaveSurfer instance
  3. Create a Minimap instance in the WaveSurfer's "ready" event callback
