With wavesurfer.js you can build anything: from a powerful DJ application to a simple audio player for your blog.

Wavesurfer.js includes support for plugins. You can use community-developed plugins to extend thwavesurfer.js core functionality or you can write your own.

Browser support

Supports all modern browsers supporting Web Audio, including Firefox, Chrome, Safari (desktop and mobile) and Opera.

Quick start

1. Include script:

Include the minified script:

(Note: you can also use a module bundler)

2. Create an HTML container

Create a container where the waveform is to appear (in HTML): 

<div id="waveform"></div>

3. Create the player

Create a wavesurfer.js instance (in JS), with the container selector along with any options you want to define (full list of options): 

var wavesurfer = WaveSurfer.create({
    container: '#waveform',
    waveColor: 'violet',
    // ... other options go here
});

4. Load the audio file

Tell wavesurfer.js which audio file to load (in JS): 

wavesurfer.load('audio.wav');

(More info: documentation of the load method)
